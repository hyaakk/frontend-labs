## ❊ Sobre o Projeto

**LessScreen** é uma aplicação web de **bem-estar digital** que transforma números em consciência. O usuário informa suas horas de tela diárias e recebe um diagnóstico honesto, fatos impactantes e dicas práticas e personalizadas para reconquistar o equilíbrio entre o mundo digital e a vida real.

Em um mundo onde a média global de uso de telas ultrapassa **7 horas por dia**, o LessScreen não condena a tecnologia — ele propõe uma relação mais intencional com ela.

O primeiro passo para mudar é **perceber**.

## ❋ Executando o Projeto

### ▸ Opção 1 — Acesso imediato (recomendado)

Sem instalação. Acesse direto pelo navegador:

**🔗 [hrsonn.github.io/lessscreen](https://hrsonn.github.io/lessscreen/)**

---

### ▸ Opção 2 — Rodar localmente

**Pré-requisitos:** Git instalado · Navegador moderno

```bash
# Clone o repositório
git clone https://github.com/hrsonn/lessscreen.git

# Entre na pasta
cd lessscreen

# Abra no navegador (Linux/Mac)
open index.html

# Abra no navegador (Windows)
start index.html
```

>  **Dica:** Use a extensão **Live Server** no VS Code para recarregamento automático durante o desenvolvimento.
